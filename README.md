# Anand Mahadev | Portfolio Website

## Core Tech Stack
-   **Frontend:** HTML5, CSS3, Vanilla JavaScript
-   **Performance:** HSL Color Palette, Custom Glassmorphism UI
-   **SEO:** Structured Data (JSON-LD), Meta Tags, Canonical Routing

## Key Features
-   **Interactive IDE Terminal:** Typewriter-style terminal for experience summaries.
-   **Game Subsytems:** Playable Snake and Brick games within the portfolio environment.
-   **Responsive Design:** Fully optimized for mobile, tablet, and high-DPI displays.
## Local Setup
1.  Clone this repository: `git clone https://github.com/anandmahadev/anandmahadev.in`
2.  Open `index.html` in your favorite browser.
3.  For full features, a local server like Live Server (VS Code Extension) is recommended.
